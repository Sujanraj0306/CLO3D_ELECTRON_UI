# CLO3D_ELECTRON_UI

A simple Electron UI for CLO3D.

## Prerequisites

- Node.js (v14+ recommended)
- npm (comes with Node.js)

## Installation

Install dependencies:

```
npm i
```

## Running the app

Start the application:

```
npm start
```

These commands will install required packages and start the Electron application.
